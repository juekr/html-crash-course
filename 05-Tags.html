Die wichtigsten Tags

(die <>-Klammern bitte denken)

Document/Head

  • !DOCTYPE: kein "echtes" HTML-Element – sagt dem Browser, welche HTML-Version er anzeigen soll
  • html: kennzeichnet das HTML-Dokument als solches (im Gegensatz zum Beispiel zu XML-Dokumenten)
  • head: Kopfbereich – alles, was hier steht, wird nicht im Browser angezeigt
  • title: Seitentitel – wird zum Beispiel in der Titelleiste des Browsers angezeigt, als Titel für Lesezeichen oder bei Google in den Suchergebnissen
  • link: Referenz auf eine Datei, deren Inhalt geladen und eingebettet werden soll (zum Beispiel CSS-Dateien)
  • meta: Meta-Informationen, die zum Beispiel von Suchmaschienen gelesen werden oder als Vorschau angezeigt werden, wenn man einen Link in einem Social Network postet
  • style: CSS-Definitionen

Seitenbereiche

  • body: Alles, was im body steht, ist direkt auf der Seite sichtbar.
  • article: Das ist eine logische Klammer, zum Beispiel für Produkte.
  • section: Das ist eine logische Klammer für Bereiche.
  • nav: Das ist eine logische Klammer für die Navigation.
  • h1, h2, h3, h4, h5, h6: Das sind hierarchisch strukturierte Überschriften. Jede Seite sollte eine H1-Überschrift haben.
  • header: Das ist eine logische Klammer für den Kopfbereich.
  • footer: Das ist eine logische Klammer für den Fußbereich.

Struktur

  • div: generischer Content-Container.
  • p: Text-Absatz
  • hr: horizintale Linie
  • pre: Code-Abschnitt
  • blockquote: Zitat/Hervorhebung
  • ol: geordnete Liste (1./2./3.)
  • ul: ungeordnete Liste (Bulletpoints)
  • li: Listen-Eintrag

Bedeutung/Semantik

  • a: Hyperlink
  • em: leichte Hervorhebung (kursiv)
  • strong: starke Hervorhebung (fett)
  • small: kleiner Text
  • del: durchgestrichener Text

Medien

  • img: Bild
  • iframe: Einbettung einer weiteren Seite
  • embed: Einbettung zum Beispiel von Youtube-Videos
  • object: Einbettung zum Beispiel von Active-X-Objekten
  • video: Video-Einbettung
  • audio: Audio-Einbettung
  • svg: Vektorgrafik

Tabellen

  • table: Tabelle
  • tbody: Datenbereich der Tabelle
  • thead: Kopfbereich der Bereiche
  • tfoot: Fußzeilen der Tabelle
  • tr: Tabellenzeile
  • th: Tabellenüberschrift(zelle)
  • td: Tabellenzelle

Formulare

  • form: Formular
  • label: Feldbeschriftung
  • input:
  • select:
  • option:
  • button:
  • textarea:

Skript

  • script: Einbingun von externen Skript-Dateien
  • noscript: wird ausgeführt, wenn der Browser kein Javascript unterstützt
  • canvas: Container für dynamisch erstellte Grafiken
